# MediaMint

Free cross-platform video & audio downloader — download and convert content from YouTube, Instagram, and more.

🔗 **Live site:** [media-mint.netlify.app](https://media-mint.netlify.app/)

## Features
- 🎥 YouTube to MP4 (up to 1080p)
- 🎵 YouTube to MP3 (320kbps)
- 📱 Instagram Reels, Stories, and video downloads
- 🚫 No ads, no login, no watermark
- 🌐 Works on any device — no software or extension needed

## Tech
Built with vanilla HTML/CSS/JS, deployed on Netlify, using serverless functions for media processing.
